# ISSUES — PINGUI

## [ВИРІШЕНО] Windows: «Java не знайдено» / mojibake in pingui-java.bat

**Симптом**

```bat
C:\...\PINGUI\java> pingui-java.bat --build
[pingui-java] ПОМИЛКА: Java не знайдено. Потрібен JDK 21.
```

**Причина:** (1) JDK 21 не в PATH; (2) баг у `pingui-java.bat`: `goto` з підпрограми завершував скрипт до Gradle (тиша в cmd).

**Рішення:** оновити репозиторій (`git pull`); встановити JDK 21 або `PINGUI_JAVA_HOME`.

---

## [ВИРІШЕНО] Gradle: «What went wrong: 25.0.3»

Gradle 8.10 не запускається під Java 25. Потрібен JDK 21 (`PINGUI_JAVA_HOME` / `JAVA_HOME`).
